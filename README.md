AreaForge — Калькулятор площади фигур на 7 языках
AreaForge — это коллекция из семи независимых реализаций калькулятора площади геометрических фигур. Каждая версия работает на своём языке программирования и предлагает удобный интерфейс для быстрого расчёта площади с расширенными возможностями.

✨ Уникальные возможности (общие для всех версий)
📐 Поддержка множества фигур:

Круг, прямоугольник, треугольник, квадрат

Трапеция, параллелограмм, ромб

Эллипс, правильный многоугольник

3D-фигуры: сфера, куб, цилиндр, конус (площадь поверхности)

🎯 Интерактивный выбор фигуры с подсказками по вводу параметров

📊 Сохранение истории вычислений (в некоторых версиях)

💾 Экспорт результатов в текстовый файл

🔄 Мгновенный пересчёт при изменении параметров (в GUI/веб)

🌍 Поддержка разных единиц измерения (см, м, дюймы)

🖥️ Интерфейсы:

Командная строка (CLI) с интерактивным меню

Веб-интерфейс для PHP
